# Netflix Content Strategy Analysis

This project analyzes Netflix's 2023 content strategy using viewership data.

## Dataset

The dataset used for this analysis is available for download here:
[Download netflix_content_2023.csv](https://statso.io/wp-content/uploads/2024/09/netflix_content_2023.csv)

**Note:** The dataset is not included in this repository. Please download it from the above link and place it in the project folder before running the analysis.

## Usage

1. Download the dataset from the link above and save it as `netflix_content_2023.csv` in the project directory.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open and run the `analysis.ipynb` notebook.

4. **Seasonal Trends**: Strategic releases during high-engagement periods
5. **Holiday Marketing**: Content alignment with key dates and events
6. **Content Optimization**: Focus on high-impact titles over volume

## Technologies Used

- **Python**: Primary analysis language
- **Pandas**: Data manipulation and analysis
- **Plotly**: Interactive data visualization
- **Jupyter Notebook**: Analysis environment

## Getting Started

### Prerequisites

- Python 3.x
- Anaconda (recommended for package management)

### Installation

1. Clone or download this repository
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Open `analysis.ipynb` in Jupyter Notebook or your preferred IDE

### Running the Analysis

1. Ensure all dependencies are installed
2. Open `analysis.ipynb`
3. Run all cells to reproduce the analysis
4. Explore the interactive visualizations
